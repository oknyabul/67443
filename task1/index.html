Использование команды ping (ping проверяет, работает ли соединение с другим устройством в сети и как быстро оно отвечает)


Использование команды nslookup для домена kubsu.ru (nslookup используется для получения IP-адреса домена или имени сервера по IP nslookup -type=A запрашивает A-запись домена, то есть его IPv4-адрес A-запись (Address Record) — это DNS-запись, которая связывает доменное имя с его IPv4-адресом. Например, example.com → 192.168.1.1)


Использование команды nslookup, чтобы узнать A-запись и MX-запись домена kubsu-dev.ru nslookup -type=MX запрашивает MX-записи домена, то есть почтовые серверы, обрабатывающие email для этого домена. MX-запись (Mail Exchange Record) — это DNS-запись, указывающая, какие почтовые серверы принимают email для домена. Например, example.com → mail.example.com.)


Использование команды whois (whois — показывает информацию о домене или IP-адресе: владельца, дату регистрации, контактные данные и DNS-серверы.)
